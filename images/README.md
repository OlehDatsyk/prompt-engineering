# Images

This folder holds visual assets referenced by the repository's documentation (banners, diagrams exported as images, screenshots of prompts in use, etc.).

Currently, diagrams in this repository are authored directly as Mermaid code blocks inside the relevant Markdown files (see the README and `prompts/programming/architecture.md` for examples), so they render natively on GitHub without needing a static image file here.

If you add static images (e.g. a repo banner or screenshots), reference them from Markdown like:

```markdown
![Alt text](images/your-image.png)
```
